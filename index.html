SPELL on Github

SPELL is a Spring Initializr CLI tool for generating projects that can be easily customized. Possible customizations include a project's dependencies, Java version, and build system or build structure.

Main commands

  • help: Display help about available commands
  • icreate: Create Spring Boot project interactively
  • create: Create a Spring Boot project non-interactively with params
  • dependency: Show details about Spring dependencies
  • param: Show values (examples) for params of the 'create' command

Installation and usage

Download appropriate zip archive with executable binary file for Windows, macOS, or Linux:

  • Windows: package-native-windows-v0.2.0.zip
  • Linux: package-native-ubuntu-v0.2.0.zip
  • macOS: package-native-macos-v0.2.0.zip

Or use curl for downloading:
curl https://github.com/spell-cli-tool/spell/releases/download/v0.2.0/package-native-windows-v0.2.0.zip -O -J -L
curl https://github.com/spell-cli-tool/spell/releases/download/v0.2.0/package-native-ubuntu-v0.2.0.zip -O -J -L
curl https://github.com/spell-cli-tool/spell/releases/download/v0.2.0/package-native-macos-v0.2.0.zip -O -J -L

Unzip and launch spell executable file. On macOS and Linux, to run a file, you need to make it executable:

chmod +x spell

Also it's possible to download zip archive with executable jar package-spell-jar-v0.2.0.zip and launch applciation with command:

java -jar spell-0.2.0.jar

All releases: https://github.com/spell-cli-tool/spell/releases
